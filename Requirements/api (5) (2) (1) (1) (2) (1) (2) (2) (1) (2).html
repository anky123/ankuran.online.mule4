post /accounts

Create an account in Salesforce

  • Request
  • Response

Headers

  • client_id: required(string)

    Client Id to be passed for authenticating the API

  • client_secret: required(string)

    Client Secret to be passed for authenticating the API

  • channel: (string)

    Consumer channel name of the API

    Example:

    Mobile
  • correlationId: required(string)

    Correlation ID used for tracking the API

Body

Media type: application/json

Type: object

Properties
  • name: required(string - pattern: (^[a-zA-Z0-9 ]*$))

    Account Name

  • email: required(string)

    User Email ID

  • number: (string)

    Account Number

  • phone: (string)

    Account's phone number

  • fax: (string)

    Account's Fax number'

  • billingAddress: (object)

    Billing Account address

    • street: (string)

      Street Name

    • city: (string)

      City Name

    • state: (string)

      State of Residence or province

    • postalCode: (string)

      Postal code or zip

    • countryIso: (string)

      Country ISO code

  • shippingAddress: (object)

    Shipping Account address

    • street: (string)

      Street Name

    • city: (string)

      City Name

    • state: (string)

      State of Residence or province

    • postalCode: (string)

      Postal code or zip

    • countryIso: (string)

      Country ISO code

Example:

{
  "name": "Sravanthi",
  "email": "abc@xyz.com",
  "number": "1011",
  "phone": "90312893293",
  "fax": "21312312",
  "billingAddress": {
    "street": "High Street",
    "city": "Hyderabad",
    "state": "Telangana",
    "postalCode": "500089",
    "country": "India"
  }
}

Body

Media type: application/json

Type: object

Properties

    Salesforce account ID

  • status: (one of 0, 1)

    Status of the result

  • message: (string)

    Result message
